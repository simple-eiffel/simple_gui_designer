Overview

simple_gui_designer is a web-based GUI specification designer built with Eiffel and HTMX. Design application screens visually using drag-and-drop, then export the specification as JSON for code generation or documentation.

Part of the simple_* ecosystem of focused, single-purpose Eiffel libraries.

Quick Start

Installation

Add to your ECF file:

<library name="simple_gui_designer" location="$SIMPLE_GUI_DESIGNER/simple_gui_designer.ecf"/>

Run the Designer

-- Start the designer server
local
    designer: GUI_DESIGNER_SERVER
do
    create designer.make (8080)
    designer.start
    -- Open http://localhost:8080 in browser
end

Features

  • Visual design canvas with 12-column grid
  • Drag-and-drop control placement
  • JSON export of specifications
  • Built with HTMX for interactivity
  • No JavaScript framework required
  • Full Design by Contract support
